Modern UK Trainset
==================

1. About
========

OpenTTE is a train set for OpenTTD, including vehicles from the Thomas The Tank Engine universe


2. Version Requirements
=======================

OpenTTD 1.10.0 or newer.
OpenTTE doesn't support TTDPatch.


3. Parameters
=============

- None

==============
